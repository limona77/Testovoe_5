
### Тестовое задание

1. Склонируй репозиторий

```bash 
  git clone https://github.com/limona77/Testovoe_5
```
2. настрой env файл и установи все зависимости

```bash 
  go mod tidy
```

3. Запусти проект

```bash 
  go run cmd/main.go
```
4. адрес сваггера
```bash 
  http://localhost:8080/swagger/index.html
  ```
